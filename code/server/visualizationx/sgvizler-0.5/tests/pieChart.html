Pie Chart: Bürgermeister und die Populationszahlen ihrer Städte (nur Städte mit > 1.000.000 Einwohner)

The element which draws the above chart:

<div id="sgvzl_example_query" 
   data-sgvizler-endpoint="http://localhost:8345/sparql/" 
   data-sgvizler-query="PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX dbpedia-owl: <http://dbpedia.org/ontology/> PREFIX dbpprop: <http://dbpedia.org/property/> SELECT DISTINCT ?mayor ?population WHERE { { ?town dbpedia-owl:leader ?mayor . ?town dbpedia-owl:populationTotal ?population . } UNION { ?town dbpedia-owl:leaderName ?mayor .  ?town dbpedia-owl:populationTotal ?population . } UNION { ?town <http://dbpedia.org/property/b%C3%BCrgermeister> ?mayor . ?town dbpedia-owl:populationTotal ?population . } FILTER (?population > 1000000) } ORDER BY DESC(?population) LIMIT 10"
   data-sgvizler-chart="gPieChart" 
   data-sgvizler-loglevel="2" 
   style="width:1200px; height:500px;"/>
